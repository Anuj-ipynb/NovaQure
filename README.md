# NovaQure

## Noise-Adaptive Hybrid AI-Quantum Framework for Intelligent Drug Discovery

NovaQure is a research-oriented hybrid AI�Quantum framework designed for accelerated and explainable drug discovery under simulated NISQ conditions.

The framework combines:

- AI-based molecular generation
- Quantum-inspired molecular evaluation
- Reliability-aware optimization
- Agentic molecular refinement
- Explainable molecular ranking
- Interactive visualization

---

# Core Features

## Molecular Generation
- JTVAE-based molecular generation
- SELFIES molecular representation
- QCBM-inspired latent exploration

## Molecular Evaluation
- chemprop Graph Neural Networks
- RDKit descriptor analysis
- VQE-based quantum simulation

## Reliability-Aware Optimization
- Noise-Adaptive Quantum Reliability Engine (NQRE)
- Quantum-guided latent refinement
- Confidence-aware evaluation

## Agentic Optimization
- Adaptive Molecular Decision Engine (AMDE)
- Iterative refinement
- Failure-aware optimization memory

## Visualization
- Interactive molecular dashboard
- Ranking visualization
- Explainable optimization workflow
